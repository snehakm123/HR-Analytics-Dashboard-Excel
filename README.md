# HR Analytics Dashboard (Excel)

## Project Overview
This interactive Excel dashboard provides a comprehensive analysis of employee data to help HR departments understand workforce trends, attrition rates, and demographic distributions. It transforms raw HR data into actionable insights for better decision-making.

### Key Metrics Tracked:
* **Total Employees:** 1,470
* **Attrition Count:** 237 (16.12% Attrition Rate)
* **Average Age:** 37
* **Job Satisfaction Rating:** 2.6 / 4.0

---




## Features & Insights
* **Demographic Breakdown:** Visualizes workforce distribution by gender (882 Male vs. 588 Female).
* **Attrition Analysis:**
    * **By Education:** Bachelor's degree holders show the highest attrition count (99).
    * **By Age Group:** Highest attrition is observed in the 25–34 age bracket.
    * **By Department:** R&D carries the largest attrition share at 56.12%.
* **Job Role Impact:** Utilizes a Tree Map to identify which roles (like Laboratory Technicians and Sales Executives) are most affected by turnover.
* **Interactive Slicers:** Allows users to filter data dynamically by **Education Field** and **Department**.

## Tools Used
* **Microsoft Excel:** Data Cleaning, Pivot Tables, and Dashboard Design.
* **Data Visualization:** Slicers, Donut Charts, Treemaps, and Bar Charts.

## How to Use
1. Download the `HR_Analytics_Dashboard.xlsx` file from this repository.
2. Open it in Microsoft Excel.
3. Click 'Enable Editing' in the yellow security bar at the top of the window.
4. Use the slicers on the bottom right to filter the entire dashboard by specific departments or education backgrounds.
